## ğŸš€ Getting Started

This repository contains the interactive technical guide itself, not the full source code of the final product. The guide is a self-contained React application.

### Prerequisites

- Node.js 18.0.0 or later
- pnpm, npm, or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/story-flow-guide.git
    cd story-flow-guide
    ```

2.  **Install dependencies**
    ```bash
    pnpm install
    ```

3.  **Start the development server**
    ```bash
    pnpm dev
    ```
    The web application will be running on `http://localhost:3000`.
